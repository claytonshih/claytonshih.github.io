🌊 波動方程與矩陣解法

撥一根弦、丟一顆石頭——波怎麼算？把空間拆成一排一排的點，你會發現「求波」其實就是「一直做矩陣運算」。

波動方程：跟前面差在「時間也是二階」

前面熱擴散的方程對「時間」只有一階導數（∂T/∂t）；波動方程對時間是二階（∂²u/∂t²）。一維（一根弦）長這樣：

一維波動方程 ∂2u∂t2 = c2 ∂2u∂x2
比喻：u 是弦上每一點的高度，c 是波速。左邊「時間二階」＝加速度，右邊「空間二階」＝弦的彎曲程度。白話就是：弦被拉得越彎的地方，回彈的加速度越大——這就是它會來回振盪、把擾動一路傳出去的原因。
還記得蛙跳法嗎？「時間二階＝由加速度決定運動」正是蛙跳法的主場。這頁時間方向用的就是中央二階差分（＝蛙跳），需要記住「前一刻」和「現在」兩張快照，才能推出「下一刻」。

兩個方向都用差分：時間 + 空間

把兩邊的二階導數各自換成「中央二階差分」（就是前面熟悉的 左 − 2×自己 + 右），一個對時間、一個對空間：

時間二階差分
∂2u∂t2 ≈ uin+1 − 2uin + uin−1Δt2
空間二階差分
∂2u∂x2 ≈ ui−1n − 2uin + ui+1nΔx2
解出下一刻
uin+1 = 2uin − uin−1 + C2(ui−1n − 2uin + ui+1n)
程式碼
next[i] = 2*u[i] - prev[i] + C2*(u[i-1] - 2*u[i] + u[i+1]);

其中 C = c·Δt / Δx 叫做 Courant 數，等一下講穩定性會用到。注意這裡出現了 n−1（前一刻）——這就是蛙跳法「需要兩張快照」的地方。

關鍵：空間差分其實是一個矩陣

你的直覺沒錯——求波的近似解，本質就是矩陣運算。把弦上所有內部點的高度疊成一個向量 u，那個「左 − 2×自己 + 右」的動作，對整條向量來說就是乘上一個三對角矩陣 K（離散拉普拉斯／剛度矩陣）：

−210000 1−21000 01−2100 001−210 0001−21 00001−2
矩陣 K：對角 −2、左右鄰居 +1、其餘為 0（其實大部分是 0，稱為稀疏矩陣）

於是整條弦「下一刻」的向量，可以一口氣寫成一行矩陣式：

向量／矩陣形式（顯式） un+1 = (2I + C2K) un − un−1
顯式（Explicit）
下一刻 = 已知向量去乘矩陣。快、簡單，但步長受 CFL 限制（見下一段）。這頁的互動都用這種。
隱式（Implicit）
未知量出現在方程兩邊，要解一個線性系統 A·un+1 = b（等於對矩陣做分解／求逆）。慢一點，但可以用大步長不爆掉。
你記得的「用矩陣求近似解」，兩種都算：顯式是「向量乘矩陣」，隱式是「解 A x = b」。工程上跑大型波場、地震、聲學、結構，多半就是在組這個稀疏矩陣、然後高效地乘它或解它。額外彩蛋：矩陣 K 的特徵向量就是駐波（simple modes），特徵值對應各模態的頻率——這也是為什麼一根弦有基頻與泛音。

互動一：撥一根弦（一維波）

用滑鼠在弦上往下拉一點再放開（或按「撥中央」），看擾動怎麼分成兩個波往兩端跑，跑到邊界就直接穿出去（吸收邊界，不反射）。每一幀就是上面那條更新式跑一次。

互動：撥弦振動（吸收邊界，可穿透）

在弦上點一下就能撥它。

⚠️ CFL 穩定條件：一維時 C = cΔt/Δx 必須 ≤ 1。把滑桿拉過 1.0，你會看到弦瞬間爆掉亂飛——這不是 bug，是「波一步跑超過一格」時，差分法必然失控的數值不穩定。這條界限就是在告訴你：時間步長不能比波穿過一格的時間還大。

互動二：丟石頭進水裡（二維波）

把弦升級成水面：空間差分從「左＋右」變成「上＋下＋左＋右」的五點差分（跟二維熱擴散同一個拉普拉斯，只是這裡時間是二階、會來回振盪而不是單向抹平）。在水面上點一下丟石頭，看同心圓波紋擴散、干涉。

互動：水面波紋（點擊丟石頭）

紅＝水面隆起，藍＝凹陷。點水面丟石頭。

波峰 靜止 波谷
⚠️ 二維的 CFL 更嚴：C ≤ 1/√2 ≈ 0.707。邊界改用吸收邊界，波紋跑到邊緣會直接穿出去、不反射回來（像開放水域而非水池壁）。這裡也加了一點點阻尼模擬能量耗散；拉高 C 一樣會看到畫面開始亂跳。

小結：波 = 一直乘（或解）矩陣

把整條線串起來看：

  1. 連續的波動方程（時間、空間都是二階）。
  2. 兩個方向都換成二階差分 → 時間用蛙跳、空間變成三對角／五點矩陣 K。
  3. 每一步就是 un+1 = (2I + C²K)un − un−1：一次矩陣運算。
  4. 想用大步長、或更複雜的邊界／材質，就改用隱式——解 Ax = b。
  5. 別忘了 CFL 條件：步長太大，波一步跨過一格，必爆。

所以「求波的近似解」跟你記得的一樣，核心就是把微分算子離散成矩陣，然後乘它或解它。真實工程（地震波、聲場、電磁 FDTD、結構振動）差別只在矩陣更大、更稀疏、邊界更講究，心法完全一樣。

想接回時間積分的直覺？回去看 數值模擬 的一維／二維熱擴散——同一個拉普拉斯矩陣，只是那邊時間一階（單向抹平），這邊時間二階（來回振盪）。
