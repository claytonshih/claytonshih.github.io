🌊 線性水波理論（Airy 波）

波浪力學怎麼從流體力學一步步推出水波方程——每條式子都附物理意義。

把波當成抽象的高度很方便，但真實的水面波其實是從流體力學推出來的。這裡用「微小振幅」假設（波高遠小於波長與水深）走一遍經典推導。你會看到一件很反直覺的事：水體內部的方程根本不會震盪，整個波動的時間動態全藏在水面那條會動的邊界上。

舞台：座標與符號

η：水面高度　·　H=2a：波高　·　L：波長　·　h：水深　·　φ：速度勢

三個假設：水不可壓縮、無黏（理想流體）、運動無旋；而且波很小（這是能「線性化」的關鍵）。無旋這條讓我們能定義一個「速度勢」φ，把三個速度分量濃縮成一個純量：u = ∂φ/∂x、w = ∂φ/∂z。

第 0 步：流體要遵守的三條律

一切從流體力學的三條基本律出發，它們對水裡每一個微小水團都成立（u、w 是水平、垂直速度，p 壓力，ρ 密度）：

① 質量守恆
（連續方程）
∇·u = ∂u∂x + ∂w∂z = 0
② 動量守恆
（Euler 方程）
ρ(∂u∂t + (u·∇)u) = −∇p + ρg
③ 無旋
∇ × u = 0

① 是「水擠不扁」：流進一個小盒子的水一定等量流出。② 就是牛頓第二定律 F=ma 用在水團上——左邊是加速度（∂u/∂t 是原地變快慢、(u·∇)u 是被水流「帶」到流速不同的地方），右邊的力只有壓力差 −∇p 和重力 ρg（理想流體，忽略黏滯）。③ 說水團不自轉——波從靜止的水面被擾動起來，無黏的水生不出渦度（Kelvin 環流定理），所以整片流場保持無旋。

「旋度」到底在量什麼？旋度 ∇×u 量的是「一小團水會不會自己打轉」。想像丟一片小葉子在水裡：如果它只隨波起伏、朝向不變，就是無旋（旋度=0）；如果它像被攪動一樣自轉，就有旋度。理想（無黏）流體沒有摩擦力去「扭」水團，加上波是從靜止起動的，所以旋度從頭到尾都是 0。

無旋送我們一個關鍵數學禮物：「任何梯度的旋度恆為零」（∇×∇φ ≡ 0）是恆等式；反過來，只要一個場無旋，就一定找得到某個純量 φ 使 u=∇φ。所以「無旋」＝「存在速度勢 φ」——這就是我們能把三個速度分量塌縮成一個 φ 來解的根本原因。

第 1 步：兩個假設，各給一條方程

無旋（③）有個漂亮的結果：任何無旋場都能寫成一個純量的梯度。於是定義速度勢 φ，把三個速度分量濃縮成一個函數（這也是後面能「只解一個 φ」的關鍵）：

速度勢（來自無旋 ③） u = ∇φ　⇒　u = ∂φ∂x , w = ∂φ∂z

再把它代進質量守恆（①），馬上得到水體內部的控制方程：

控制方程（來自質量守恆 ①） ∇²φ = ∂2φ∂x2 + ∂2φ∂z2 = 0
物理意義：Laplace 方程說 φ 在水體內部「處處平滑、沒有源也沒有匯」。它不含時間、不會自己震盪——水波的節奏完全由水面決定，內部只是被動地把水面的擾動傳遞下去。這也是水波跟弦振動最不一樣的地方。

第 2 步：把動量方程積分成 Bernoulli（壓力從這裡進來）

動量方程（②）看起來兇，但無旋讓它每一項都能寫成「某個東西的梯度」，於是能一口氣對空間積分。三個改寫：

對流項
(u·∇)u = ∇(½|u|2) − u×(∇×u)　無旋 ⇒ 後項 = 0
局部加速
∂u∂t = ∂∂t(∇φ) = ∇(∂φ∂t)
壓力＋重力
−1ρ∇p + g = −∇(pρ + gz)　ρ 為常數、重力位能 gz

三項都是梯度，Euler 方程整條收成 ∇[ ∂φ/∂t + ½|∇φ|² + p/ρ + gz ] = 0。一個量的梯度處處為零，代表它在整個空間是同一個值（最多隨時間變），把這常數記作 C(t)，就得到非定常 Bernoulli 方程：

非定常 Bernoulli（動量方程的積分） ∂φ∂t + 12|∇φ|2 + pρ + gz = C(t)

逐項的物理意義：∂φ/∂t 是流場隨時間變化帶來的加速度貢獻、½|∇φ|² 是動能（每單位質量）、p/ρ 是壓力、gz 是重力位能。整條就是沿流場的能量帳本——也是待會把「水面壓力＝大氣壓」翻譯成 φ 條件的橋樑。

第 3 步：三個邊界條件（各自的物理）

海底 z=−h
（不透水）
∂φ∂z = 0　底部垂直速度 w=0，水不鑽進海床
液面·運動學
（幾何）
w = ∂η∂t + u∂η∂x　在 z=η 上
液面·動力學
（受力）
∂φ∂t + 12|∇φ|2 + gη = 0　在 z=η 上（Bernoulli 取 p=0）

運動學條件是純幾何：水面上的水團永遠是水面，不會突然潛進水裡或飛出去，所以它的垂直速度必須剛好等於水面本身的升降速度。動力學條件是受力：水面正上方是空氣，水面壓力必須等於大氣壓（設為 0），把 p=0 代進 Bernoulli 就得到上式。

線性化（微小振幅假設）：波很小 ⇒ 速度很小 ⇒ 帶「平方」的項（½|∇φ|²、u·∂η/∂x）都是「小×小」可丟掉；而且水面幾乎貼著 z=0，就把邊界條件從 z=η 移到 z=0 上算。兩條液面條件於是簡化成 ∂φ/∂z = ∂η/∂t（運動學）與 ∂φ/∂t + gη = 0（動力學）。

第 4 步：合併成一條液面條件

兩條線性化的液面條件都含 η。把動力學式 ∂φ/∂t + gη = 0 對時間微分，再用運動學式 ∂η/∂t = ∂φ/∂z 把 η 消掉，就剩一條只含 φ 的式子：

合併後的自由液面條件（z=0） ∂2φ∂t2 + g ∂φ∂z = 0
物理意義：這其實就是水面的「F = ma」——∂²φ/∂t² 對應水面上下振盪的慣性（加速度），g·∂φ/∂z 對應重力回復力。重力想把凸起的水面拉平、慣性又讓它衝過頭，一來一回就振盪成波。

第 5 步：代入行進波，解出「垂直結構」

猜一個往 +x 前進的波：η = a·cos(kx − ωt)、φ = Z(z)·sin(kx − ωt)（k = 2π/L 波數、ω = 2π/T 角頻率）。代進 Laplace 得 Z″ − k²Z = 0，通解是 cosh、sinh 的組合；再用海底條件 Z′(−h)=0 砍掉一半，得到：

速度勢 φ = a gω · cosh k(z+h)cosh kh · sin(kx − ωt)

物理意義：cosh k(z+h) 在水面 (z=0) 最大、往海底 (z=−h) 遞減，就是「擾動越深越弱」的數學形狀；它在 z=−h 的斜率為 0，正好滿足「水不鑽進海床」。下面的動畫會直接讓你看到這個遞減。

第 6 步：頻散關係（整段的結晶）

把 φ 記成 φ = (ag/ω)·F(z)·sin(kx−ωt)，垂直因子 F(z) = cosh k(z+h) / cosh kh，在水面 F(0)=1、斜率 F′(0)=k·tanh(kh)。代進第 4 步的液面條件 ∂²φ/∂t² + g·∂φ/∂z = 0（在 z=0）逐項算：

時間二階
∂2φ∂t2 = −ω2φ　→ 在 z=0：−ω2·agω·sin
垂直一階 ×g
g∂φ∂z = g·agω·F′(0)·sin = g·agω·k·tanh(kh)·sin
相加 = 0
(−ω2 + g k·tanh(kh))·agω·sin(kx−ωt) = 0

括號外的 (ag/ω)·sin(kx−ωt) 不可能處處為零（否則就沒有波了），所以只能是括號＝0，直接得到波浪力學最核心的一條式子：

頻散關係 Dispersion Relation ω2 = g k · tanh(kh)

物理意義：左邊 ω² 是振盪的快慢（慣性）；右邊 gk 是重力回復力（g 提供回復、k 越大代表水面越彎、回復越急）；tanh(kh) 是「有多少水在跟著動」的修正——深水這層很厚（tanh→1），淺水被海底卡住、能動的水變少（tanh→kh）。它把「波震多快（ω）」跟「波多長（k）」綁在一起，相速度 c = ω/k = √[(g/k)·tanh(kh)]，於是有兩個經典極限：

深水 h/L > ½
tanh → 1，ω² = gk，c = √(g/k) = gT/2π。波長越長跑越快 → 有頻散。
淺水 h/L < 1/20
tanh(kh) ≈ kh，c = √(gh)。只跟水深有關 → 無頻散，正是波動模擬頁那條 ∂²η/∂t² = gh·∂²η/∂x²。

第 7 步：看得見的結果——水粒子怎麼動

把 φ 微分得速度、再積分回位移，會發現水粒子其實在繞橢圓：水平半徑 ∝ cosh k(z+h)、垂直半徑 ∝ sinh k(z+h)。拉動「水深」滑桿，看橢圓怎麼從深水的圓一路壓扁成淺水的扁橢圓：

互動：波浪下的水粒子軌道

直覺：深水裡，往下不到一個波長水就幾乎沒感覺了（潛水艇不怕表面浪）；淺水裡，連貼著海底的水都在前後來回滑動——這就是為什麼浪進到淺灘會「摸到底」、變陡、最後捲起來碎掉。

左：深水——圓形軌道隨深度指數縮小、很快消失　｜　右：淺水——扁橢圓，越靠海底越只剩水平滑動
