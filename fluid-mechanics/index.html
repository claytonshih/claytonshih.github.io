💧 流體力學基礎：三條律從哪來

在推水波之前，先把地基打好——用「跟著水團看」的角度，把連續方程、Euler 與無旋，從最基本的物理一步步蓋出來。

這頁在做什麼？大多數水波推導一開頭就丟出流體的三條律（連續方程、Euler 動量方程、無旋）當作已知。這頁反過來，從最基本的物理把這三條律一條條蓋出來——包含那個最常讓人卡住的 (u·∇)u 對流項。先打好這層地基，之後看 線性水波理論 的每一步都會有根據。

先講一個關鍵切換：跟著水團看

流體力學所有「怪怪的」符號，幾乎都來自同一個選擇：你要站在原地看水流過去，還是跟著某一團水一起漂？兩種看法各有用處，但牛頓定律 F=ma 只對「一團固定的水」成立——因為 m 要是同一團物質。所以我們得先學會怎麼在「原地座標」裡，描述「跟著水團」的變化。這個橋樑就是隨體導數，也是這頁的主角。

第 1 塊地基：把水看成「場」，不看分子

水其實是一顆顆亂撞的分子，但要一顆顆追根本不可能。連續體假設說：只要我們看的尺度遠大於分子間距，就可以把水想成「處處都有值」的連續場——每一點都有密度 ρ(x,t)、速度 u(x,t)、壓力 p(x,t)。

左：一顆顆亂撞的分子（看不出規律）　→　右：抹成連續的速度場，每點一個箭頭

比喻：報紙照片放大看是一堆小網點，退遠一點就變成連續的臉。我們不追每個網點（分子），只描述「這一區大致多黑」（密度、速度）——這就是把流體當成場。

第 2 塊地基：隨體導數 D/Dt（對流項的來源）

想像一條河，下游河道收窄。水擠不扁，同樣的水量要擠過更小的截面，就得流得更快（為什麼？下一節的連續方程會證明）——所以越往下游、水流越快。現在丟一片葉子順流而下：就算河水穩定、每個定點的流速都不隨時間變，葉子仍會一路加速。它的加速度拆成兩份：

① 局部項
（定點就在變）
∂u∂t　上游把水龍頭越開越大那份；水流穩定時為 0
② 對流項
（被水流帶走）
(u·∇)u = u∂u∂x　被自己帶進更窄、更快的河段那份
合計＝隨體導數
DuDt = ∂u∂t + (u·∇)u　＝水團真正的加速度

對流項為什麼是「速度 × 梯度」的形狀

第①項很直觀。第②項要看清楚，得跟著葉子實際走一小步 Δt。這一步裡有兩件互相獨立的事同時發生——時間過了 Δt，位置也往下游移了 Δx。把它們對流速的貢獻分別算出來、再加起來，u·∂u/∂x 的形狀就自然長出來了。

葉子沿著「越下游越快」的斜坡走了 Δx，就自動撿到一截速度差 ∂u/∂x·Δx——這截加速就是對流項的來源。

走了多遠
Δx = u·Δt　速度 × 時間
① 時間過了 Δt
就算待在原地，流速也可能變（水龍頭開大）：Δu₁ = ∂u∂t·Δt
② 位置移了 Δx
移到更窄、本來就更快的河段：Δu₂ = ∂u∂x·Δx
兩者相加
Δu = ∂u∂t·Δt + ∂u∂x·Δx　就是多變數的全微分（連鎖律）
除以 Δt
DuDt = ∂u∂t + ∂u∂x·ΔxΔt = ∂u∂t + u∂u∂x　代入 Δx/Δt = u
用單位想最快：對流項 =（每公尺快多少 ∂u/∂x，(m/s)/m = 1/s）×（流速 u，m/s）= m/s² ＝加速度。所以它必然是相乘：河道收得越急、或流得越快，被「載」出來的加速就越大。三維只是把「往哪走」拆成三個方向：(u·∇)u。

下面自己拉滑桿：調「河道收窄」與「水龍頭」，看漂流的葉子（藍）怎麼一路加速，而釘在原地的流速計（灰）在水流穩定時讀數不變。葉子多出來的加速，全是對流項。

互動：定點流速不變，水團卻在加速

局部項 ∂u/∂t
0.0
對流項 u·∂u/∂x
3.0
Du/Dt（葉子的加速度）
3.0

單位是「m/s² 示意」。水流穩定時（水龍頭 = 0）灰針不動，但葉子仍一路加速——那全是對流項。

怎麼讀 (u·∇)u 這個記號

前面一步步推出來的是一維的 u·∂u/∂x。真實的水是三維，葉子可以被帶往任何方向，緊湊的寫法就是 (u·∇)u。把它拆成兩層看就通了：

∇（梯度算子）
∇ = (∂∂x, ∂∂y, ∂∂z)　一排偏導數
u·∇
（點積→純量運算子）
u∂∂x + v∂∂y + w∂∂z　「被水流帶著走，每秒撿到多少變化」
作用在 u 上
(u·∇)u　對速度每個分量各做一次 → 一個向量＝對流加速度

那為什麼是「點積」？走一小步時，位移 Δx = u·Δt 是個有方向的向量。任何量 Q 撿到的變化，是梯度點上你實際移動的方向——只有「沿著梯度」的那截算數，沿著等值線走 Q 不變：

ΔQ = ∇Q·Δx：位移在 ∇Q 方向的投影（粉）才改變 Q；沿等值線的分量不改變 Q。點積就是在挑出「有效的那截」。

一句話記住 u 為什麼出現兩次：外面那個 u——往哪走、走多快（水流把你載去哪）；裡面那個 u——被載著走的量，剛好是速度本身。一維時你只沿 x 走，點積就塌回前面逐步推出的那條 u·∂u/∂x。
這條 Du/Dt 就是要放進 F=ma 的加速度。它的向量形式是 ∂u/∂t + (u·∇)u；右邊的 (u·∇)u 正是 水波理論頁 ② 用到的對流項——由「水團被自己的流動帶到流速不同的地方」而生。下一節先把質量守恆補上（順便證明為什麼河道窄、流速就快），再用 F=ma 把整條動量方程組起來。

第 3 塊地基：質量守恆 → 連續方程

抓一個固定不動的小方盒（控制體積），盯著水從各面進出。質量不會憑空生滅：盒內質量增加的速率，一定等於「淨流入」。寫成式子就是連續方程：

質量守恆（連續方程） ∂ρ∂t + ∇·(ρu) = 0

水幾乎壓不扁（不可壓縮，ρ 為常數），上式就塌成非常乾淨的一條——流進來多少，就要流出去多少：

不可壓縮 ⇒ 體積守恆 ∇·u = ∂u∂x + ∂w∂z = 0

下面調左右兩面的流速，看盒子「爆滿」還是「抽乾」。只有當進 = 出（散度 ∇·u = 0）時，密度才穩得住——這正是不可壓縮的意思。

互動：進多少，就得出多少

第 4 塊地基：牛頓第二定律 → Navier–Stokes → Euler

對「一團水」用 F=ma。加速度我們剛推過，就是隨體導數 Du/Dt。那力有哪些？盯著一個水立方，只有三種：

壓力差
−∇p　左右面壓力不一樣 → 把水團往低壓推
重力
ρg　整團水的重量
黏滯力
μ∇²u　鄰居水層的「摩擦拖曳」，想把速度抹平

把「質量×加速度 = 三種力」寫出來，就是流體力學的中心方程——Navier–Stokes 方程：

Navier–Stokes（黏性流體） ρ(∂u∂t + (u·∇)u) = −∇p + ρg + μ∇²u

水波推導裡我們假設理想流體——水的黏性很小，先把黏滯項 μ∇²u 丟掉（μ→0），Navier–Stokes 就退化成更簡潔的 Euler 方程：

Euler 方程（理想流體）＝ 水波頁的 ② ρ(∂u∂t + (u·∇)u) = −∇p + ρg
丟掉黏性合理嗎？對「大尺度的水波」來說，黏性只在很薄的邊界層才重要，主體幾乎無摩擦。這也是為什麼水波能傳很遠都不太衰減。糖漿、蜂蜜那種就不能丟——它們的 μ 很大。

第 5 塊地基：渦度與無旋 → 速度勢 φ

最後一條律是「無旋」。渦度 ω = ∇×u 量的是「一小團水會不會自己打轉」。反直覺的是：水流走直線也可能在自轉！下面切換兩種流場，看那個小十字槳輪轉不轉：

互動：葉片會不會自己轉？

關鍵直覺：剪切流裡，槳輪上緣的水比下緣快，一快一慢就把它扭轉起來——即使每條流線都是直的。均勻流上下一樣快，只把槳輪平移、不轉。理想流體從靜止被擾動、又沒有黏性去「扭」水團，所以渦度從頭到尾是 0（Kelvin 環流定理）——這就是水波「無旋」的來歷。

無旋送我們一份大禮：數學上「任何梯度的旋度恆為零」（∇×∇φ ≡ 0），反過來只要流場無旋，就一定存在一個純量 φ 使——

無旋 ⇒ 存在速度勢 u = ∇φ　⇒　u = ∂φ∂x , w = ∂φ∂z

三個速度分量瞬間濃縮成一個純量 φ。再把它代回連續方程 ∇·u = 0，就得到水體內部那條著名的控制方程 ∇²φ = 0（Laplace 方程）——而這，正好就是水波推導的起跑線。

小結：三條律到齊，可以推水波了

回頭看，我們只用了三個守恆／假設，就蓋出水波頁「憑空丟出」的那三條律：

質量守恆
→ 連續方程 ∇·u = 0（① 連續方程）
牛頓 F=ma
→ Navier–Stokes → 丟黏性 → Euler（② 動量方程）
理想＋靜止起動
→ 無旋 ∇×u = 0 → 速度勢 φ（③ 無旋）
地基打好了，接著就把這三條律套進「會動的水面」邊界，一路推到頻散關係 ω²=gk·tanh(kh)。前往 線性水波理論（Airy 波）。想先看波怎麼用電腦一步步算，則看 波動模擬。
